# 🏦 Bank Management System (JavaScript OOP)

## 📌 Overview
A simple bank management system built using JavaScript, HTML, and CSS.  
This project demonstrates how Object-Oriented Programming (OOP) concepts can be used to build real-world applications.

---

## 🚀 Features
- Create a bank account  
- Deposit money  
- Withdraw money  
- View account details  
- Track total number of accounts (static property)  
- Clean and interactive UI  

---

## 🧠 Concepts Used
- Classes and Objects  
- Constructor and `this` keyword  
- Inheritance (SavingsAccount)  
- Static properties and methods  
- DOM Manipulation  
- Event Handling  

---

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript  

---

## 📂 Project Structure:-
project/
├── index.html

---

## ⚙️ How It Works
- User creates an account using name and initial balance  
- System stores account data using JavaScript classes  
- Users can deposit and withdraw money through UI buttons  
- Account details are displayed dynamically on the screen
